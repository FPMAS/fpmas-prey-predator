Base Grass constructor.

Initializes only growing_rate, grown and grow_count_down fields: GridCellBase specified fields, such as the cell location, are left undefined and must be initialized afterwards. This is automatically handled by the JSON serialization process, what allows this constructor to be safely used in the to_json() method below.
